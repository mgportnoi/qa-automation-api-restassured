# QA Automation API - RestAssured

Projeto de automação de testes de API utilizando Java + RestAssured + JUnit.

## Objetivo

Demonstrar conhecimentos em:

* Testes de API
* Métodos HTTP (GET, POST)
* Validação de status code
* Validação de corpo da resposta
* Estrutura de testes automatizados
* Uso de JUnit

## Tecnologias

* Java
* Maven
* RestAssured
* JUnit 5

## Cenários Automatizados

* Listagem de usuários (GET)
* Criação de usuário (POST)
* Validação de status code 200 e 201
* Validação de payload de resposta

## Execução

Executar via Maven:
mvn test

## Conceitos Aplicados

* Requisições HTTP
* Status Codes
* Assertions
